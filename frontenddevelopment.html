Frontend Development | Winkelstraat

Voor het vak Frontend Development kreeg ik de opdracht om een website na te maken van een website die je zelf mocht uitkiezen, waarbij wel enige uitdaging in zat. Het doel van dit vak was om deze website in nette vanilla HTML, CSS & Javascript te schrijven, dit was niet het geval bij deze website.

Concept

Werk in minimaal twee pagina's van de bestaande website uit, waarbij er nette HTML, CSS & Javascipt geschreven moet zijn. Het bevat minimaal één uitgewerkte micro-interactie. Tot slot moet de website toegangelijk zijn voor iedereen.

Bekijk mijn winkelstraat website

Micro-interactie

De micro-interactie die ik heb gemaakt is het toevoegen van een item aan de favoriete lijst. De gebruiker drukt op het hartje, er komt een 1 in het hartje boven aan komt te staan.
